# 프론트엔드 작업 규칙

이 프로젝트는 순수 HTML/CSS/JS 단일 파일 구조이다.

## 코드 스타일

- 기존 CSS 변수(`:root`)를 우선 사용한다. 새 색상을 임의로 만들지 않는다.
- 기존 클래스 네이밍 패턴을 따른다.
- 한글 콘텐츠의 폰트는 기존 설정(Noto Sans KR)을 유지한다.
- 반응형 디자인: 기존 미디어 쿼리 브레이크포인트를 따른다.

## 변경 원칙

- HTML 구조 변경 시, 기존 섹션 순서와 계층을 유지한다.
- CSS 추가 시, 기존 파일의 끝에 추가한다. 기존 규칙을 수정하지 않는다.
- JavaScript 추가 시, 기존 이벤트 핸들러와 충돌하지 않는지 확인한다.
